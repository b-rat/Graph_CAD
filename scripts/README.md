# Scripts

Command-line scripts for training, evaluation, and data processing.

## Example Scripts (to be created)

- `train.py` - Train graph autoencoder models
- `evaluate.py` - Evaluate trained models on test data
- `preprocess_cad.py` - Convert CAD files to graph representations
- `generate.py` - Generate new CAD models using trained generative models
- `visualize.py` - Visualize graphs and model outputs

## Usage

Scripts should use `argparse` or `click` for command-line arguments and load configuration from `config/` files.
